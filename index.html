Glacier

Glacier is a type annotation extension to Java that enforces transitive class immutability.

View on GitHub Download .zip Download .tar.gz

Glacier helps keep your immutable data immutable.

Replication package

Glacier has been shown to help users express and enforce immutability, preventing bugs. If you want to replicate the study, you can use the same materials we did. You'll need to install IntelliJ (we used IntelliJ IDEA Community 2016.2) and JDK 1.8 separately. Replication Package
